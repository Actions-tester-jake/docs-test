= rpk container status
:rpk_version: v23.1.6 (rev cc47e1ad1)

Get a local container's status.

== Usage

[,bash]
----
rpk container status [flags]
----

== Flags


[cols=",,",]
|===
|*Value* |*Type* |*Description*
|-h, --help |- |Help for status.
|-v, --verbose |- |Enable verbose logging (default: false).
|===
